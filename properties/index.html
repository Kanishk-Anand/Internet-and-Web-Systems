CSS Properties Description

-Kanishk Anand

Property Description Attribute
Width Defines width of element px / %
Height Defines height of an element px / %
backgound Css properties group to manipulate collective background properties url (' ')
padding Defines space between element's border and element's objects inside it px / %
margin Defines position of an element relative to its parent url (' ')
border Used to give border to an element px / em
border-color Used to manipulate color of border of an element colorname/ #hexadecimal
font-family Used to manipulate font of text FontName eg. Georgia
font-size Used to manipulate size of text px / em
box-shadow Used to give shadow to a div px / em / %
border-image used to set image as border url ('path_to_image')
display Used to manipulate display of an element none,inline,block,table,inline-block
clip-path Used to manipulate shape of a div/section clip-path function eg. polygon()
border-radius Used to manipulate border edges of a div px / %
nth-child() Used to select specific child inside parent element positive integer
float Used to position the div relative to parent left,right,none
clear Used to specify where not to position div relative to parent left,right,both,none
cursor Used to specify shape of mouse-cursor specific to an element pointer,hourclass,hand,help etc.
